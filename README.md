# ColabSecomAnalysis
colab과 Python을 이용하여 반도체 공정 데이터 이상 예측하기 + 머신러닝을 이용한 분석

## 💻 기술 스택

- Jupyter Notebook
- python
- seaborn
- matplotlib
- pandas
- numpy
- StandardScaler
- XGBoost 


## ✨ 참고사항

출처: https://archive.ics.uci.edu/ml/datasets/SECOM

이 데이터 세트에는 반도체 공정에서 측정된 센서 데이터(총 591개 센서)와 실패여부(Pass/Fail)가 포함되어 있다.
